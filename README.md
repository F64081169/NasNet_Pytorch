# NasNet_Pytorch

Using pre-trained nasnet model inference a photo via pytorch

## Usage 

```
mkdir data
```
Next, put the photo you want to predict with under the /data directory.
```
pip install pretrainedmodels 
```
```
python nasnet_test.py
```
